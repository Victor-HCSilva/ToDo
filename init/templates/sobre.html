Sobre este app

Este é um aplicativo simples de lista de tarefas (To-Do List) desenvolvido com o framework web Django, que segue o padrão Model-View-Controller (MVC) para uma estrutura robusta e escalável. Ele permite aos usuários criar, visualizar, editar, marcar como concluído e excluir anotações. O foco principal foi criar uma aplicação funcional e intuitiva, demonstrando a capacidade do Django em gerenciar o backend e a lógica de negócios.

Para o frontend e a estilização da interface do usuário, foi utilizado o framework CSS Bulma. Bulma é um framework moderno e responsivo, baseado em Flexbox, que facilita a criação de layouts bonitos e adaptáveis a diferentes tamanhos de tela. Ele contribui para a experiência de usuário limpa e eficiente que você encontra neste aplicativo.
